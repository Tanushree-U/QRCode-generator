#QRCode-generator

##Description:
This project generates qrcode for the link provided using python

##Languages and tools:
Python
QRCode library

##IDE used:
Visual Studio Code

##How to run
1. Install qrcode library
2. Run qrcode.py
3. QR code image will be generated as myqr.png
